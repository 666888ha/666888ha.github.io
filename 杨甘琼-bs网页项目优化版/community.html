Great ways to learn VS Code

Outside of the official documentation, there are additional videos, tutorials, and repositories we think are cool! Use these to learn how to make the most out of VS Code.

If you've found additional tutorial content especially helpful, please let us know where to find it by filling out this short form.s

1
Visual Studio Code - Pluralsight course

John Papa runs through this course to show you how to debug, get IntelliSense, use the multi-cursor, and work in a variety of languages like ASP.Net, JavaScript, and TypeScript.

Duration 4 hr hours
2
Learning Visual Studio Code - LinkedIn Learning course

Reynald Adolphe helps you get started with VS Code. He explains how to prepare for environments like ASP.NET, and shows how to create your first .NET app and Node app.

Duration 1.3 hr hours
3
Make Visual Studio Code Your Editor - Scotch.io course

Created by Chris Sevilleja, this course covers the main features of VS Code and covers language specific uses for JavaScript, Angular, Node.js, and PHP.

Duration 2.5 hr hours
4
Learn Ionic 3 From Scratch - Udemy course

Paul Halliday shows you how to create a cross platform mobile application with Ionic, Angular, TypeScript and Firebase all in VS Code.

Duration 9.5 hr hours

Tip: You can get a free 3 month subscription of Pluralsight with Visual Studio Developer Essentials.
